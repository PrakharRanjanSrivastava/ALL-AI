# ALL AI - Full-Stack AI SaaS Platform

## 🚀 Overview
A comprehensive suite of AI-powered micro-tools designed to streamline content creation and document analysis.

## 🛠️ Tech Stack
- **Frontend:** React, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL (Prisma ORM)
- **Authentication:** Clerk Auth
- **AI Integration:** Google Gemini API, Clipdrop API
- **Cloud Storage:** Cloudinary

## ✨ Key Features
- **AI Content Generation:** Automated article and blog title generation using Google Gemini.
- **Resume Analyzer:** PDF processing and analysis tool.
- **Image Processing:** Background and object removal via Clipdrop and Cloudinary.
- **Subscription Model:** Tiered access (Free vs. Premium) managed via Stripe and Clerk.
